1. Скачай чертежи

→ Выберите модель: «Уютный» (1–2 кота) или «Семейный» (4–5 котов).
→ Скачай ZIP-архив → внутри: DXF-чертежи (для резки) + PDF-инструкция.
→ Всё бесплатно! И даже регистрация не нужна!

"Уютный"  /  "Семейный"

2. Изготовь детали

→ Купи 2 листа фанеры толщиной 6 мм
→ передай DXF-файлы в мастерскую для резки (лазер/фрезер).
→ Обработай края наждачкой
→ Пропитай влагостойким составом (для улицы).

3. Собери домик

→ Склей ножки, стены, крышу.
→ Закрепи миски герметиком — чтобы они не стали трофеем вандалов!
→ Поставь домик на ровное место и зови гостей 🐾
